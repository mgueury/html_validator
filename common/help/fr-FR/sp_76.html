Sp 76 : Elément ... indéfini

Cause :

Vous utilisez un élément qui n'est pas défini dans le type de document que vous déclarez.
N'utilisez pas de balises en majuscules en XHTML.
Attention en particulier aux balises propriétaires.

Exemple :

<object width="400" height="300" type="application/x-shockwave-flash" data="movie.swf">
<embed name="movie" value="movie.swf" />
</object>
<object width="400" height="300" type="application/x-shockwave-flash" data="movie.swf">
<param name="movie" value="movie.swf" />
</object>

Solution :

Corrigez l'élément pointé.
