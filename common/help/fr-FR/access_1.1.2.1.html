Accessibilité : [1.1.2.1] : texte alternatif ou lien descriptif manquant sur un élément img

Priorité 1

Cause :

Si le texte alternatif (alt) ne peut pas décrire la fonction d'une image ou le but de celle-ci avec une courte description, proposez une description plus complète par l'intermédiaire de l'attribut longdesc. Vous devez également ajouter un lien descriptif (d-link) car beaucoup de navigateurs ne supportent pas l'attribut longdesc. L'avantage d'inclure une description longue dans un document actuel est que la description demeurera accessible quand de nouveaux navigateurs n'implémenteront plus les liens descriptifs.

Exemple :

<img src="graphique-camembert.jpg" alt="Graphique camembert des dépenses fédérales">

Solution :

Insérez un texte alternatif.
