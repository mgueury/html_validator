Accessibilité : [6.2.2.1] : les équivalences textuelles nécessitent une mise à jour (applet).

Priorité 1

Cause :

A chaque fois que vous modifiez le contenu dynamique, les équivalents textuels doivent être mis à jour. Vous utilisez une appliquette (applet) qui peut inclure du contenu dynamique, si c'est le cas, maintenez les équivalents textuels à jour dans une section <noscript> par exemple.

Exemple :

<applet><!-- applet délivrant du contenu dynamique --></applet>
<applet><p>Un contenu dynamique présente ...</p><applet>

Solution :

Les équivalences pour le contenu dynamique doivent être mises à jour.
