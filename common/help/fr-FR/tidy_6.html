Aide de tidy - Tidy [6] : Il manque un </...>

Cause :

Une balise n'est pas fermée. La plupart des balises HTML doivent être fermées.

Exemple :

<head>
<title>
hello
</head>
<head>
<title>
hello
</title>
</head>

Solution :

Fermez la balise tel que nécessaire. Attention toutefois aux balises auto-fermées, qui doivent adopter la structure suivante (avec attributs dans la balise):

  • <base />
  • <meta />
  • <link />
  • <hr />
  • <br />
  • <basefont />
  • <param />
  • <img />
  • <area />
  • <input />
  • <isindex />
  • <col />
