Accessibilité : [12.1.1.2] : Le titre de la <frame> n'est pas valide (espaces)

Priorité 1

Cause :

Il faut donner un titre à chaque <frame> (cadre) ou <iframe> d'un document. Les titres des frames (cadres) doivent être signifiants (c'est-à dire permettre d'identifier le contenu et/ou les liens pour la navigation). Ne pas utiliser pour titre des noms de fichiers (par exemple : title="cadre1.html") ou un texte de substitution (par exemple : title="insérer ici un titre de cadre").

Exemple :

<frame title=" "></frame>
<frame title="courte description du cadre"></frame>

Solution :

Insérez un titre qui ne soit pas composé uniquement d'espaces.
