Accessibilité : [1.1.1.3] : texte alternatif alt douteux (taille du fichier)

Priorité 1

Cause :

Toutes les images exigent un texte équivalent, mais l'attribut alt doit également répondre à certains critères autrement il sera considéré douteux. Par exemple, le texte de l'attribut alt peut être considéré comme inacceptable si la taille du fichier est employée au lieu d'un texte équivalent réel (ex : alt="650 octets").

Exemple :

<img src="gifimage.gif" alt="34K bytes"/>
<img src="gifimage.gif" alt="intérêt de l'image dans la page"/>

Solution :

Indiquez un texte alternatif (attribut 'alt') qui donne une indication sur le sens de la présence de l'image dans la page.
