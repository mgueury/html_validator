Accessibilité : [1.1.5.1] : texte alternatif manquant sur un élément object

Priorité 2

Cause :

Il faut synchroniser les alternatives équivalentes avec une présentation. Les éléments object doivent contenir un contenu alternatif pour s'assurer que l'information est accessible à ceux qui ne peuvent pas accéder aux objets graphiques ou au multimédia. Le contenu alternatif devrait contenir les même informations ou atteindre le même objectif que l'objet lui-même. Vous pouvez fournir un lien vers une description ou l'insérer directement dans le document.

Exemple :

<object classid="myclass.class" title="Ceci est un mauvais objet"></object>

Solution :

Il faut synchroniser les alternatives équivalentes (i.e. les sous-titres ou descriptions auditives d'une piste visuelle) avec la présentation.
