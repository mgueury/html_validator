Accessibilité : [1.1.6.1] : Le texte de description d'un fichier sonore est manquant (wav)

Priorité 1

Cause :

Les pages Web qui contiennent des fichiers audio doivent également fournir une description des fichiers sonore où reproduisez le nom des fichiers. Si le fichier audio contient une grande quantité d'information, établissez un lien dans sur ce document qui pointera vers une description complète du fichier sonore. Cette technique est appropriée pour reproduire les paroles d'une chanson et les extraits de discours.

Exemple :

<a href="clap.wav">The sound of one hand clapping (wav)</a>

Solution :

Insérez le texte de description.
