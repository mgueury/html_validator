Sp 141 : ID ... déjà défini

Cause :

L'attribut 'id' a deux fois la même valeur dans le document. L'identifiant (la valeur de cet attribut) doit être unique dans un document.

Exemple :

<span id="name1">...<span id="name1">
<span id="name1">...<span id="name2">

Solution :

Modifiez l'un des ID.
