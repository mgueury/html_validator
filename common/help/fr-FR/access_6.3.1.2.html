Accessibilité : [6.3.1.2] : Les éléments programmables doivent être testés (object)

Priorité 1

Cause :

Vérifiez que le contenu reste accessible lorsque les éléments programmables sont désactivés ou ne sont pas supportés. Si un objet (<object>) est utilisé pour générer du contenu, fournissez un contenu équivalent dans l'élément 'object' lui-même.

Exemple :

<object><!-- l'object --></object>
<object><!-- l'object --> Cet objet montre...</object>

Solution :

Insérez un équivalent textuel dans l'élément 'object'.
