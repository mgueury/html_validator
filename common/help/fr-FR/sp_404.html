Sp 404 : Le caractère ... est le premier caractère d'un délimiteur mais il est interprété comme donnée

Cause :

Plusieurs causes possibles

  • Vous avez essayé d'écrire "<" dans les données de votre page. Il faut l'écrire "&amp;lt;" dans le HTML.
  • Vous avez utilisé une éperluette sous la forme "&". Il faut écrire "&amp;" dans le HTML.
  • Vous avez oublié de fermer les guillemets

Exemple :

<p>Une balise commence par : < </p>
<p>Une balise commence par : &amp;lt; </p>

Solution :

Modifiez les caractères spéciaux incriminés.
