Sp 407 : balise ouvrante NET-enabling n'est pas immédiatement suivie d'une balise fermante

Cause :

Ce message apparaît lorsqu'il y a une erreur dans la manière dont une balise auto-fermée est fermée. La syntaxe correcte est : <... /> (attention à l'espace avant la barre oblique).

Solution :

Corrigez la balise.
