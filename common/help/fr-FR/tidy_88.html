Aide de tidy - Tidy [88] : le caractère ";" n'est pas autorisé

Cause :

le caractère ";" n'est pas autorisé dans la liste de spécification des attributs.
Ce message peut apparaître lorsque du code Javascript est mal encapsulé.

Exemple : 

<script language="javascript">
for(x=0;x<item.length;x++){
    //
}
</script>          
<script language="javascript">
//<![CDATA[ 
for(x=0;x<item.length;x++){
    //
}
//]]> 
</script>

Solution :

Encapsulez correctement le code Javascript.
