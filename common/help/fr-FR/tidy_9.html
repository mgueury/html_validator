Aide de tidy - Tidy [9] : balise imbriquée superflue

Cause :

Vous avez écrit deux balises identiques imbriquées simplement. Ce n'est pas une erreur HTML.

Exemple :

<strong><strong>Voici le texte</strong></strong>
<strong>Voici le texte</strong>

Solution :

Supprimer les ... superflus. La deuxième balise n'ajoute aucune information.
