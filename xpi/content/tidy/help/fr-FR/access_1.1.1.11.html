Accessibilité : [1.1.1.11] : texte alternatif manquant dans <img> (puce)

Priorité 1

Cause :

Toutes les images doivent avoir un équivalent textuel, y compris celles qui identifient les éléments d'une liste. Si une image est utilisée comme puce, elle devrait être signalée par alt="*" ou bien alt="bullet".

Exemple :

<img src="star.jpg" width="5" height="5">

Solution :

Insérez un texte alternatif.
