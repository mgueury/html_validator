Accessibilité : [1.1.1.10] : texte alternatif alt douteux (texte trop long)

Priorité 1

Cause :

Toutes les images exigent un texte équivalent, mais l'attribut alt doit également répondre à certains critères autrement il sera considéré douteux.
Si le texte alternatif dépasse les 150 signes ou espaces, utilisez l'attribut longdesc avec un lien descriptif (d-link) pour décrire l'image.

Exemple :

<img src="gifimage.gif" alt="0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789 0123456789"/>
<img src="gifimage.gif" alt="intérêt de l'image dans la page"/>

Solution :

Indiquez un texte alternatif (attribut 'alt') qui donne une indication sur le sens de la présence de l'image dans la page.
