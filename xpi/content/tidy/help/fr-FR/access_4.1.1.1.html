Accessibilité : [4.1.1.1] : Il faut identifier clairement les changements de langue du texte

Priorité 1

Cause :

Vous utilisez des changements de langue sans les identifier avec les balises et attributs appropriés.

Exemple :

<p> Il faut utiliser le formulaire européen
<span lang="en"> Fiscal Residence Form 111 </span>
pour cette déclaration.</p>

Solution :

Il faut identifier clairement les changements de langue du texte d'un document et de tous les équivalents textuels, comme les légendes. Lorsqu'une page comporte plus d'une langue, il faut s'assurer que les changements de langue sont identifiés avec l'attribut « lang ».
