Sp 387 : Séparateur 'S' dans une déclaration de commentaire

Cause :

Ce message peut apparaître si vous avez des commentaires consécutifs et que l'un d'eux est mal fermé. La syntaxe correcte pour le commentaire est :<!-- commentaire-->

Solution :

Vérifiez et corrigez les commentaires.
