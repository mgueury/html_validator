Accessibilité : [7.2.1.1] : Retirer la balise blink

Priorité 2

Cause :

Vous utilisez des balises qui créent du mouvement ou du clignotement dans les pages.

Exemple :

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN" "http://www.w3.org/TR/REC-html40/loose.dtd">
<html lang="en">
<head>
<title>alerte 1.0/7.2.1</title>
</head>
<body>
<p>Here is a word of <blink>blinking</blink> text.</p>
</body>
</html>

Solution :

Retirer les balises <blink> ou <marquee>.

  • W3C WAI : http://www.w3.org/WAI/
  • Techniques HTML pour le WAI 1.0: http://www.w3.org/TR/WCAG10-HTML-TECHS/
