Sp 73 : Fin de l'élément ... qui n'est pas complet

Cause :

Vous avez oublié de placer un élément fils obligatoire, ainsi l'élément pointé n'est pas complet.

Exemple :

<ul>Ma liste</ul>
<ul><li>Ma liste</li></ul>

Solution :

Assurez-vous d'avoir bien déclaré tous les éléments fils obligatoires. Les éléments qui ont des fils obligatoires sont :

  • HTML (nécessite HEAD et BODY)
  • HEAD (nécessite TITLE)
  • UL (nécessite LI)
  • OL (nécessite LI)
  • DL (nécessite DT ou DD)
  • SELECT (nécessite OPTION)
  • TABLE (nécessite TBODY ou TR)
  • THEAD, TFOOT, TBODY (nécessitent TR)

Références :
