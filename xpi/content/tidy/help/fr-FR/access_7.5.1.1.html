Accessibilité : [7.5.1.1] : Il ne faut pas utiliser de balises pour rediriger les pages automatiquement

Priorité 2

Cause :

Vous utilisez un mécanisme de redirection.

Exemple :

<meta http-equiv="refresh" content="http://www.foo.com/bar.html" />

Solution :

Tant que les agents utilisateurs ne fournissent pas la possibilité d'arrêter les auto-redirections, il ne faut pas utiliser de balises pour rediriger les pages automatiquement. À la place, il faut configurer le serveur pour effectuer les redirections.
