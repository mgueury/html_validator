Sp 63 :Les données ne sont pas autorisées ici

Cause :

Vous utilisez des données à un emplacement dans le fichier où elle ne sont pas admises.
Vous avez par exemple placé des données dans le corps du fichier (entre les balises <body>) sans conteneur ("p" par exemple). Vous avez peut être omis les guillemets autour d'une valeur d'attribut contenant les caractères particuliers "%" et "/".

Exemple :

<body>
hello <a href="test.html">world</a>
</body>
<body>
<p> hello <a href="test.html">world</a>
</p> </body>

Solution :

Ajoutez un conteneur ("p" pour paragraphe par exemple) ou placez la valeur de l'attribut entre guillemets.
